# 🛠 LioS Game Dumper (Root Only)

![LioS](https://img.shields.io/badge/Game-Last_Island_of_Survival-orange)
![Platform](https://img.shields.io/badge/Platform-Android-green)
![Root](https://img.shields.io/badge/Required-Root-red)

Специализированный дампер для выгрузки данных из игры **Last Island of Survival** (Global & Lite).

## 📌 Описание
Инструмент предназначен для получения дампов памяти (offsets, classes, strings) в среде Android. Работает на уровне системы как Root-модуль, обеспечивая стабильный доступ к процессу игры.

## ✨ Особенности
* 🎯 **Полная совместимость:** Поддерживает актуальные версии LioS Global и LioS Lite.
* ⚡ **Root Power:** Работает через Magisk / KernelSU / APatch.
* 🛠 **Чистый дамп:** Оптимизирован под архитектуру игры для минимизации мусора в выходных файлах.
* 🚫 **No Multi-game:** Только для LioS. Под другие игры не адаптирован.

## 🚀 Установка
1. Скачайте `.zip` архив с релизом.
2. Установите его через **Magisk** или **KernelSU**.
3. Перезагрузите устройство.
4. Запустите игру, дампер начнет работу автоматически или по команде (зависит от вашей реализации).

## ⚠️ Дисклеймер
Данный инструмент создан исключительно в образовательных целях и для помощи в реверс-инжиниринге. Автор не несет ответственности за блокировки аккаунтов или другие последствия использования данного софта.
